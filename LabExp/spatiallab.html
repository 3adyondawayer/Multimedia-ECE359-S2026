Spatial Filtering

This explanation page introduces the main spatial filters used in digital image processing, with emphasis on MATLAB built-in functions first and manual implementation second. The page is designed to help students quickly decide which filter to choose, how to call it in MATLAB, and what effect to expect on the image.

Linear filters Nonlinear filters Smoothing Sharpening MATLAB-focused

Reference note

Example images in the lab should be taken from Gonzalez and Woods, Digital Image Processing, 3rd edition. That gives the lab a familiar visual reference and keeps the examples aligned with standard textbook figures used in class.

The explanations below begin with the built-in MATLAB workflow students are most likely to use in the lab, then connect that workflow to the underlying manual implementation.

Lab roadmap

Spatial filters compute a new pixel value using a neighborhood around the current pixel. Some filters are linear and use weighted sums, while others are nonlinear and rely on order statistics such as median, minimum, or maximum.

Recommended MATLAB workflow

  • Read the image with imread.
  • Convert to grayscale when the task is defined for a single channel.
  • Apply the built-in filtering function first.
  • Repeat the same idea manually to verify understanding.
  • Display original and processed outputs side by side using subplot or tiledlayout.
Core teaching philosophy: students should first identify the right built-in function, understand its parameters, and observe the visual effect. After that, they should reproduce the same behavior manually using convolution, correlation, neighborhood extraction, or order-statistic logic.

Linear filters

Linear spatial filtering combines neighboring pixels using a kernel or mask. In MATLAB, this is usually done with imfilter, fspecial, conv2, or related functions.

Average (Box) Filter

Each pixel is replaced by the arithmetic mean of its neighborhood. This smooths noise and small detail, but it also blurs edges.

Smoothing · Linear
Typical 3×3 mask: (1/9) × ones(3,3)

How to use it

Use fspecial('average',[m n]) to build the mask, then apply it with imfilter(I,H,'replicate'). Increase the mask size when you want stronger blur, but expect more loss of fine detail and sharper edge softening.

Best use

Choose it when you need a simple low-pass filter, want easy MATLAB syntax, or need a quick baseline smoothing result for comparison.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
H = fspecial('average', [3 3]);
J = imfilter(I, H, 'replicate');

figure;
subplot(1,2,1); imshow(I); title('Original');
subplot(1,2,2); imshow(J); title('Average Filter');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
I = double(I);
H = ones(3,3) / 9;
J = conv2(I, H, 'same');
J = uint8(J);

figure;
imshow(J); title('Manual Average Filter');
  • Built-in focus: emphasize fspecial('average') and imfilter.
  • Normalization is essential so brightness does not increase.
  • Use it as the simplest reference smoothing filter in comparisons.

Weighted Average Filter

Nearby pixels still contribute to the output, but the center receives a larger weight. This reduces noise with less blurring than a plain box filter.

Smoothing · Linear
Example mask: (1/16) × [1 2 1; 2 4 2; 1 2 1]

How to use it

Define the weighted mask manually, normalize it so the coefficients sum to 1, then call imfilter. This is often introduced as a stepping stone between a plain average filter and a Gaussian filter.

Best use

Choose it when you want mild smoothing but would like to preserve central structure better than a uniform box filter.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
H = [1 2 1; 2 4 2; 1 2 1] / 16;
J = imfilter(I, H, 'replicate');

figure;
imshow(J); title('Weighted Average');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
I = double(I);
H = [1 2 1; 2 4 2; 1 2 1] / 16;
J = conv2(I, H, 'same');
J = uint8(J);
  • Center weighting usually preserves important structure slightly better.
  • Because the mask is symmetric, correlation and convolution give the same result.
  • Useful when you want students to see why all averages are not equally good.

Gaussian Filter

Gaussian smoothing performs weighted averaging where nearby pixels contribute more according to a bell-shaped distribution. It is widely used because it suppresses noise smoothly and naturally.

Smoothing · Linear

How to use it

Create the kernel with fspecial('gaussian',[m n],sigma) and apply it with imfilter. Control the blur using both the kernel size and the value of sigma; larger values produce stronger smoothing.

Best use

Choose it when you want smooth noise reduction, more natural blur, or preprocessing before derivative-based filters such as Sobel or Laplacian.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
H = fspecial('gaussian', [5 5], 1.0);
J = imfilter(I, H, 'replicate');

figure;
imshow(J); title('Gaussian Filter');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
I = double(I);
[x, y] = meshgrid(-2:2, -2:2);
sigma = 1.0;
H = exp(-(x.^2 + y.^2) / (2*sigma^2));
H = H / sum(H(:));
J = conv2(I, H, 'same');
J = uint8(J);
  • Discuss the effect of kernel size and standard deviation sigma.
  • Larger kernels and larger sigma values produce stronger blur.
  • Often the preferred smoothing operator before edge detection.

Laplacian Filter

The Laplacian is a second-derivative operator that highlights rapid intensity change. It is useful for sharpening and edge emphasis.

Sharpening · Linear
Common mask: [0 -1 0; -1 4 -1; 0 -1 0]

How to use it

Use fspecial('laplacian',alpha) or define the mask manually, compute the Laplacian response, then combine it with the original image for sharpening. Convert to double first so subtraction and addition behave correctly.

Best use

Choose it when you want to emphasize fine detail or sharpen intensity transitions, especially after light smoothing to reduce noise sensitivity.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
H = fspecial('laplacian', 0);
L = imfilter(double(I), H, 'replicate');
J = uint8(double(I) - L);

figure;
imshow(J); title('Laplacian Sharpening');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
I = double(I);
H = [0 -1 0; -1 4 -1; 0 -1 0];
L = conv2(I, H, 'same');
J = I - L;
J = uint8(max(min(J,255),0));
  • Explain the difference between displaying the Laplacian and using it for sharpening.
  • Second derivatives are sensitive to noise, so pre-smoothing can help.
  • Good for showing why sharpening is different from simple edge extraction.

Sobel Filter (Gradient)

Sobel computes an approximation of the first derivative and is commonly used to measure edge strength and direction.

Sharpening / Edge Detection · Linear

How to use it

Use fspecial('sobel') for one direction, transpose it for the other, then compute horizontal and vertical responses and combine them into a magnitude image. Use mat2gray when preparing the magnitude for display.

Best use

Choose it when you need edge strength, gradient direction intuition, or a strong classroom example of first-derivative filtering.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
Gx = fspecial('sobel');
Gy = Gx';
Sx = imfilter(double(I), Gx, 'replicate');
Sy = imfilter(double(I), Gy, 'replicate');
Mag = sqrt(Sx.^2 + Sy.^2);
Mag = uint8(255 * mat2gray(Mag));

figure;
imshow(Mag); title('Sobel Magnitude');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
I = double(I);
Gx = [-1 0 1; -2 0 2; -1 0 1];
Gy = [-1 -2 -1; 0 0 0; 1 2 1];
Sx = conv2(I, Gx, 'same');
Sy = conv2(I, Gy, 'same');
Mag = sqrt(Sx.^2 + Sy.^2);
Mag = uint8(255 * mat2gray(Mag));
  • Teach horizontal and vertical gradients separately before combining them.
  • Its coefficients contain mild smoothing compared with simpler derivative masks.
  • Great for boundary emphasis and edge interpretation.

Prewitt Filter

Prewitt is another first-derivative operator similar to Sobel, but with a simpler coefficient pattern. It is useful for comparing gradient masks.

Sharpening / Edge Detection · Linear

How to use it

Use fspecial('prewitt') or define the masks manually, compute directional responses, then combine them into a magnitude image. The workflow is very similar to Sobel, which makes it ideal for side-by-side teaching.

Best use

Choose it when you want a simple gradient operator and when the goal is to compare first-derivative masks rather than maximize smoothing.

Show MATLAB built-in style 
I = imread('your_image.png');
I = rgb2gray(I);
Gx = fspecial('prewitt');
Gy = Gx';
Px = imfilter(double(I), Gx, 'replicate');
Py = imfilter(double(I), Gy, 'replicate');
Mag = sqrt(Px.^2 + Py.^2);
Mag = uint8(255 * mat2gray(Mag));
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
I = double(I);
Gx = [-1 0 1; -1 0 1; -1 0 1];
Gy = [-1 -1 -1; 0 0 0; 1 1 1];
Px = conv2(I, Gx, 'same');
Py = conv2(I, Gy, 'same');
Mag = sqrt(Px.^2 + Py.^2);
Mag = uint8(255 * mat2gray(Mag));
  • Compare Prewitt and Sobel on the same image for learning value.
  • Prewitt is slightly simpler, while Sobel weights the center row or column more.
  • Useful for teaching the concept of gradient masks clearly.

High-boost Filter

High-boost filtering sharpens an image by adding scaled detail back to the original. It is a generalization of unsharp masking.

Sharpening · Linear
High-boost idea: J = A·I - blurred(I), where A > 1

How to use it

First blur the image, subtract the blurred version from the original to obtain a detail mask, then add a scaled version of that mask back to the image. In MATLAB, this is easy to organize using fspecial, imfilter, and a gain parameter.

Best use

Choose it when the image is slightly blurred and you want stronger controllable sharpening than Laplacian alone, especially when you want to tune the boost amount.

Show MATLAB built-in style 
I = imread('your_image.png');
I = rgb2gray(I);
I = double(I);
H = fspecial('gaussian', [5 5], 1.0);
B = imfilter(I, H, 'replicate');
A = 1.5;
J = A*I - B;
J = uint8(max(min(J,255),0));

figure;
imshow(J); title('High-boost Result');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
I = double(I);
H = ones(5,5) / 25;
B = conv2(I, H, 'same');
mask = I - B;
k = 1.2;
J = I + k * mask;
J = uint8(max(min(J,255),0));
  • Start from the intuition of blurred image plus detail mask.
  • Explain the relation between unsharp masking and high-boost filtering.
  • Increasing the boost also increases the risk of noise amplification.

Nonlinear filters

Nonlinear filters do not compute a weighted sum. Instead, they examine the values inside the neighborhood and choose an output using ranking or extrema, which makes them very useful for impulse noise and local neighborhood analysis.

Median Filter

The median filter replaces the center pixel with the median of the neighborhood values. It is especially effective against salt-and-pepper noise because it removes isolated spikes without averaging across edges too aggressively.

How to use it

Call medfilt2(I,[m n]) on a grayscale image. Small windows such as 3×3 are common because they remove impulse noise while keeping edges more intact than simple averaging.

Best use

Use it first when the noise type is salt-and-pepper or impulse noise.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
J = medfilt2(I, [3 3]);
imshow(J); title('Median Filter');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
P = padarray(I, [1 1], 'replicate');
J = zeros(size(I), 'uint8');
for r = 1:size(I,1)
    for c = 1:size(I,2)
        block = P(r:r+2, c:c+2);
        J(r,c) = median(block(:));
    end
end

Max Filter

The max filter assigns the largest value in the neighborhood to the output pixel. It tends to brighten small bright regions and can expand light details.

How to use it

Use ordfilt2(I,k,domain) with k equal to the last ordered element in the neighborhood. For a 3×3 window, that means k = 9.

Best use

Choose it when you want to emphasize or expand bright structures in a local neighborhood.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
J = ordfilt2(I, 9, ones(3,3));
imshow(J); title('Max Filter');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
P = padarray(I, [1 1], 'replicate');
J = zeros(size(I), 'uint8');
for r = 1:size(I,1)
    for c = 1:size(I,2)
        block = P(r:r+2, c:c+2);
        J(r,c) = max(block(:));
    end
end

Min Filter

The min filter assigns the smallest value in the neighborhood to the output pixel. It tends to darken small dark regions and can expand dark details.

How to use it

Use ordfilt2(I,1,ones(3,3)) for a 3×3 neighborhood. This makes the smallest local intensity become the output at each pixel.

Best use

Choose it when you want to emphasize dark structures or study the effect of local minima.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
J = ordfilt2(I, 1, ones(3,3));
imshow(J); title('Min Filter');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
P = padarray(I, [1 1], 'replicate');
J = zeros(size(I), 'uint8');
for r = 1:size(I,1)
    for c = 1:size(I,2)
        block = P(r:r+2, c:c+2);
        J(r,c) = min(block(:));
    end
end

Range Filter

The range filter computes the difference between the maximum and minimum values in the neighborhood. This highlights local variation and often responds strongly near edges and textured areas.

How to use it

Call rangefilt(I,true(3)) or any other neighborhood shape you want to study. Display the result with imshow(J,[]) because the output often needs scaling for visualization.

Best use

Choose it when your goal is to measure local variation, texture, or edge activity rather than smooth the image.

Show MATLAB built-in function 
I = imread('your_image.png');
I = rgb2gray(I);
J = rangefilt(I, true(3));
imshow(J, []); title('Range Filter');
Show manual implementation 
I = imread('your_image.png');
I = rgb2gray(I);
P = padarray(I, [1 1], 'replicate');
J = zeros(size(I), 'uint8');
for r = 1:size(I,1)
    for c = 1:size(I,2)
        block = P(r:r+2, c:c+2);
        J(r,c) = max(block(:)) - min(block(:));
    end
end

Which filter to use

This comparison is meant to help students make a quick practical choice before coding. The best filter depends on whether the goal is smoothing, sharpening, edge emphasis, impulse-noise removal, or local variation analysis.

Situation Recommended filter Why it fits
You need the simplest smoothing baseline Average filter It is the easiest low-pass filter to understand and implement.
You want smoothing with slightly better center preservation Weighted average filter The center pixel has higher influence than its neighbors.
You want smoother and more natural blur Gaussian filter Its weights decay smoothly with distance and it works well before derivative filters.
You want detail emphasis through second derivatives Laplacian filter It responds strongly to rapid intensity change and is commonly used for sharpening.
You want gradient-based edge strength Sobel filter It gives directional derivatives and a strong practical edge magnitude image.
You want a simpler gradient-mask comparison Prewitt filter It is easy to interpret and compare with Sobel.
You want controllable sharpening from a blurred image High-boost filter It sharpens by adding scaled detail back to the original image.
You have salt-and-pepper noise Median filter It removes isolated outliers without averaging them across the neighborhood.
You want to expand bright details Max filter Each output becomes the local maximum.
You want to expand dark details Min filter Each output becomes the local minimum.
You want local contrast or texture activity Range filter It measures the difference between local maximum and local minimum.

Built-in first, manual second

Topic Built-in emphasis Manual follow-up Main idea students should learn
Average / weighted / Gaussian fspecial + imfilter Construct kernel and use conv2 Smoothing is neighborhood averaging with proper normalization.
Laplacian / Sobel / Prewitt Use ready masks and compute filtered response Write the masks explicitly and combine gradient terms manually Sharpening and edge extraction depend on derivatives.
High-boost Blur first, then subtract or add detail mask Compute mask directly from the original and blurred image Sharpening can be understood as preserving low-frequency content while amplifying detail.
Median / max / min / range medfilt2, ordfilt2, rangefilt Loop through each neighborhood with padding Nonlinear filters rely on ordered values, not weighted sums.

Common notes and pitfalls

Border handling

When filtering near the image boundary, MATLAB needs a padding rule. In teaching examples, 'replicate' is often the easiest choice because it avoids artificial dark borders caused by zero padding.

Data type awareness

For manual computations, converting to double before convolution is safer. After processing, clamp values to the valid range and convert back to uint8 before display when appropriate.

Interpretation first

Students should never stop at “the code works.” They should connect each filter to its effect on noise, edges, detail, and local intensity variation.
